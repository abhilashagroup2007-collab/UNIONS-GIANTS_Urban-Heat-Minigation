## Bharatiya Antariksh Hackathon 2026

### Problem Statement
Urban areas are increasingly experiencing elevated temperatures due to the Urban Heat Island (UHI) effect caused by rapid urbanization, reduced vegetation cover, and extensive built-up infrastructure.

### Proposed Solution
An AI-powered Urban Heat Mitigation Decision Support System that utilizes satellite imagery, environmental data, GIS information, and Machine Learning models to:

- Identify Urban Heat Hotspots
- Predict Heat-Risk Levels
- Analyze Contributing Factors
- Recommend Location-Specific Cooling Strategies
  
### Features
- Heat Vulnerability Mapping
- Heat Risk Prediction
- AI Cooling Recommendation Engine
- Sustainability Dashboard
- Decision Support System
